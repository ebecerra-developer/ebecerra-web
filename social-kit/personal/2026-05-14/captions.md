## Producción técnica (referencia)

- Fuentes: `_templates/carruseles/4-demos/*.html`, `_templates/stories/*.html`, `_templates/posts/lo-que-no-hago.html`
- Capturas demos: `personal/captures/demo-*.png` (re-utilizables para futuras stories)
- Scripts node: `_templates/scripts/capture-demos.mjs`, `render-statics.mjs`, `record-chatbot.mjs`
- Playwright local en `social-kit/node_modules` (no toca el monorepo)
- ffmpeg via `ffmpeg-static` npm package

Para regenerar todo:
```bash
cd social-kit/_templates/scripts
node capture-demos.mjs      # re-capturar demos vivas
node render-statics.mjs     # re-renderizar PNGs
node record-chatbot.mjs     # re-grabar vídeo
```
